# Cifrados y Criptografía

## Generación de Llaves

Este repositorio contiene scripts para la generación de llaves y cifrados en ASCII. Los scripts permiten crear llaves de tamaño fijo y dinámico, así como generar nuevos cifrados basados en estas llaves.
